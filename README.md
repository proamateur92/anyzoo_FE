## 🎯 애니쥬의 핵심기능
- 로그인/회원가입, 번거로운건 싫잖아요, 간편한 카카오/구글 소셜로그인!
- 자랑하개 에서 귀여운 반려동물 카테고리별로 구경하기!
- 홈화면에 매일/매주 좋아요 랭킹 소개
- 릴스 에서 숏동영상으로 더더욱 생동감있게!
- 함께하개 에서 우리동네 같이 산책할 사람, 공구할 사람 찾아보기
- 커뮤니티 에서 반려인들끼리 모여모여~ 자유롭게 소통하기
